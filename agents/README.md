## 🚀 Как использовать агентов

### Запрос агента:
```
@PyQt-Agent создай окно настроек для AI-Помощника

Требования:
- Вкладки: Общие, LLM, Интерфейс, О программе
- Тёмная тема
- Валидация полей
- Сохранение в YAML
```

### Ответ агента:
```python
# Агент генерирует код...
class SettingsWindow(QDialog):
    def __init__(self, parent=None):
        super().__init__(parent)
        self.setWindowTitle("⚙️ Настройки")
        self.setMinimumSize(600, 400)
        self.setup_ui()
```

## 🔄 Рабочий процесс с агентами

```
1. Ты: Описываешь задачу
   ↓
2. Диспетчер: Определяет нужного агента
   ↓
3. Агент: Создаёт код/структуру
   ↓
4. Проверка: Код ревью (Test-Agent)
   ↓
5. Деплой: DevOps-Agent запускает CI/CD
   ↓
6. Документация: Docs-Agent обновляет wiki
```
