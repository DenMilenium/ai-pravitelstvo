## 🎯 Как работает компания

### Процесс разработки фичи:

```
1️⃣ PM-Agent создаёт ТЗ
         ↓
2️⃣ UX-Agent рисует прототип
         ↓
3️⃣ UI-Agent создаёт дизайн
         ↓
4️⃣ Frontend-Agent + Backend-Agent пишут код
         ↓
5️⃣ CodeSec-Agent проверяет безопасность
         ↓
6️⃣ AutoTest-Agent пишет тесты
         ↓
7️⃣ Test-Agent проверяет функционал
         ↓
8️⃣ DevOps-Agent деплоит
         ↓
9️⃣ TechWriter-Agent обновляет документацию
```

### Пример запроса:

**Ты говоришь:** _"Сделай форму входа с валидацией"_

**Агенты работают:**
- 🎨 **UI-Agent**: Рисует макет формы
- 💻 **Frontend-Agent**: Кодирует React компонент
- ⚙️ **Backend-Agent**: API авторизации
- 🔒 **CodeSec-Agent**: Проверяет SQL-инъекции, XSS
- 🧪 **AutoTest-Agent**: Тесты валидации
- 📝 **TechWriter-Agent**: Документация API

## 🚀 Быстрый старт

### Дать задачу конкретному агенту:
```bash
# Дизайн
python agents/ui_agent.py "Создай дизайн формы входа"

# Тестирование
python agents/test_agent.py "Напиши тесты для API"

# Безопасность
python agents/codesec_agent.py "Проверь код на уязвимости"
```

### Дать задачу команде:
```bash
# Диспетчер сам выберет нужных агентов
python agent_dispatcher.py "Сделай полноценный чат"
```
