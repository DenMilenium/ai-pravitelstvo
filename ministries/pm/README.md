# 📋 Министерство Проектного Управления

> *"Порядок в хаосе разработки"*

## 🎯 Миссия

Управление проектами, внедрение методологий разработки, контроль качества, стандарты кодирования.

## 📋 Обязанности

### 🔄 Методологии
- Agile / Scrum
- Kanban для исследований
- Git Flow / Trunk Based

### ✅ Контроль качества
- Code review процессы
- CI/CD стандарты
- Тестирование (unit, integration, e2e)

### 📜 Стандарты
- Структура проектов
- Именование
- Документирование

## 📁 Ресурсы

```
ministries/pm/
├── 📁 templates/          # Шаблоны проектов
│   ├── desktop-app/       # Шаблон десктопного приложения

│   ├── onboarding.md      # Ввод новых сотрудников

│   └── release.md         # Процесс релиза

```
