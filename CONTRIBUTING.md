# 🤝 Как внести вклад в AI Правительство

Спасибо за интерес к нашему проекту! Вот как стать частью правительства:

## 🚀 Быстрый старт

1. **Форкните** репозиторий
2. **Клонируйте** свой форк
3. **Создайте ветку** для ваших изменений
4. **Внесите изменения** и закоммитьте
5. **Создайте Pull Request**

## 🏛️ Выбор министерства

Каждое министерство отвечает за свою область:

- 🖥️ **Десктоп** → Приложения для ПК
- ☁️ **Облако** → Облачные сервисы
- 🔬 **Исследования** → ML/R&D
- 📋 **PM** → Процессы и стандарты
- 🔧 **Инфраструктура** → DevOps
- 👥 **HR** → Документация и onboarding

## 📝 Стандарты

### Коммиты
```
[министерство] краткое описание

Подробное описание изменений
```

Примеры:
- `[desktop] добавлен поиск по файлам`
- `[cloud] оптимизация API`
- `[research] новый датасет`

### Code Style
- Python: PEP 8 + Black
- Go: gofmt + golangci-lint
- TypeScript: ESLint + Prettier

### Тесты
- Минимум 80% покрытие
- Все тесты должны проходить
- Новые фичи = новые тесты

## 🎖️ Получение звания

| Звание | Условия |
|--------|---------|
| 🥉 Стажёр | 1+ merged PR |
| 🥈 Специалист | 5+ merged PR |
| 🥇 Эксперт | 20+ merged PR + менторство |
| 💎 Министр | Руководство министерством |

## 💬 Общение

- GitHub Issues — баги и фичи
- GitHub Discussions — обсуждения
- Telegram — быстрые вопросы

## 📜 Кодекс

1. Уважай других участников
2. Будь конструктивен в review
3. Документируй сложное
4. Помогай новичкам
