## 🚀 Установка

```bash
# Клонирование
git clone https://github.com/DenMilenium/court-department.git
cd court-department

# Установка
npm install

# Запуск
npm run dev
```

Откройте [http://localhost:3000](http://localhost:3000)

### Docker

```bash
docker-compose up -d
```
