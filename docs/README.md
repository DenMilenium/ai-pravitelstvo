# AI Правительство

Добро пожаловать в цифровое государство AI! 🤖🏛️

## С чего начать?

1. Прочитайте [главный README](./README.md)
2. Ознакомьтесь с [CONTRIBUTING.md](./CONTRIBUTING.md)
3. Выберите министерство и начните!

## Быстрые команды

```bash
# Показать справку
make help

# Запуск тестов
make test

# Проверка кода
make lint

# Статистика
make stats
```

## Структура

- `ministries/` — министерства (отделы)
- `docs/` — документация
- `.github/` — CI/CD конфигурации
