# 🔒 Безопасность

## Ответственное раскрытие

Если вы обнаружили уязвимость безопасности, пожалуйста:

1. **НЕ** создавайте публичный issue
2. Отправьте письмо на security@ai-gov.ru
3. Дайте нам время на исправление (90 дней)

## Меры безопасности

- Все секреты хранятся в переменных окружения
- Регулярное обновление зависимостей
- Сканирование уязвимостей в CI/CD
