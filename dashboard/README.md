# 🎛️ AI Правительство - Dashboard

Панель управления для AI IT-компании "Цифровое Правительство"

## 🚀 Быстрый старт

### 1. Установка зависимостей

```bash
cd dashboard
pip install -r requirements.txt
```

### 2. Запуск

```bash
python app.py
```

### 3. Открыть в браузере

```
http://localhost:5000
```

## 🔐 Данные для входа

- **Логин**: `admin`
- **Пароль**: `admin123`

⚠️ **Важно**: Сразу после первого входа смените пароль!

## 📁 Структура

```
dashboard/
├── app.py                 # Flask backend
├── requirements.txt       # Зависимости
├── dashboard.db          # SQLite база данных
├── static/               # CSS, JS, изображения
└── templates/            # HTML шаблоны
    ├── login.html        # Страница входа
    ├── dashboard.html    # Главный дашборд
    └── profile.html      # Профиль пользователя
```

## ✨ Функции

- 🔐 **Аутентификация** — защищённый вход с паролем
- 🔒 **Смена пароля** — возможность изменить пароль
- 📊 **Статистика** — общая информация об агентах
- 🤖 **Управление агентами** — список всех 80 агентов
- 📈 **Категории** — распределение по министерствам
- 🕐 **Активность** — логи запусков агентов

## 🛡️ Безопасность

- Пароли хранятся в хешированном виде (Werkzeug)
- Сессии с секретным ключом
- Защита от CSRF через Flask session
- Валидация всех форм
