### ШАГ 4: Настройка переменных окружения GitHub

```bash
# Открыть файл службы
nano /etc/systemd/system/ai-dashboard.service
```

**Добавить в секцию [Service] (после WorkingDirectory):**

```ini
[Service]
Type=simple
User=root
WorkingDirectory=/var/www/ai-pravitelstvo/dashboard

# ДОБАВИТЬ ЭТИ СТРОКИ (вставь СВОЙ токен):
Environment="GITHUB_TOKEN=ghp_ВАШ_ТОКЕН_ЗДЕСЬ"
Environment="GITHUB_REPO=DenMilenium/ai-pravitelstvo"

ExecStart=/var/www/ai-pravitelstvo/dashboard/venv/bin/gunicorn -w 4 -b 127.0.0.1:5000 app:app
Restart=always
```

**Сохранить:** `Ctrl+O`, `Enter`, `Ctrl+X`

### ШАГ 7: Проверка работы

Открой в браузере: `http://194.67.66.120/login`

#### Тест 1: Логин
- Логин: `admin`
- Пароль: `admin123`
- ✅ Должен войти в Dashboard

#### Тест 2: Создание проекта
1. Перейди в "📋 Проекты"
2. Нажми "➕ Новый проект"
3. Введи название и ТЗ (например: "Создать лендинг для кофейни")
4. Нажми "Создать проект"
5. ✅ Должен создаться проект с задачами

#### Тест 3: Проверка прогресса
1. Открой проект
2. Посмотри на ползунок прогресса
3. Должно показывать: "0 из N задач (0%)"
4. ✅ Прогресс должен быть виден (не завис на 10%)

#### Тест 4: Выполнение задачи
1. Найди задачу со статусом "pending"
2. Нажми кнопку "▶️ Выполнить"
3. Подожди 2-3 секунды
4. ✅ Статус должен измениться на "done"
5. ✅ Прогресс должен увеличиться!

#### Тест 5: Сборка ZIP
1. Выполни все задачи (или несколько)
2. Нажми "📦 Собрать ZIP"
3. ✅ Должен появиться архив

#### Тест 6: GitHub синхронизация (опционально)
1. В проекте нажми "🔗 Синхронизировать с GitHub"
2. Подожди
3. Проверь свой GitHub репозиторий
4. ✅ Должен появиться Project с задачами

#### Тест 7: Чат с агентом
1. Перейди в "🤖 Агенты"
2. Нажми на Frontend Agent
3. Напиши "Привет" или "Что ты умеешь?"
4. ✅ Агент должен ответить!

## 🐛 Если что-то не работает

### Проблема: "Internal Server Error"
```bash
# Смотри логи
journalctl -u ai-dashboard -n 50 --no-pager
```

### Проблема: Не обновляется прогресс
```bash
# Перезапусти сервис
systemctl restart ai-dashboard

# Проверь что код обновился
cd /var/www/ai-pravitelstvo
git log --oneline -1
# Должно быть: e0b85ec или новее
```

### Проблема: GitHub не синхронизируется
```bash
# Проверь переменные окружения
systemctl show ai-dashboard --property=Environment

# Должно показать GITHUB_TOKEN и GITHUB_REPO
```
